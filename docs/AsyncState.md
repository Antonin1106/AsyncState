#### timePassed

```cpp
bool timePassed(unsigned long interval)
```

Defined in AsyncState.h:71

Checks if the specified interval has passed since the last reset.

#### Parameters
* `interval` Duration in milliseconds to check. 

#### Returns
`true` if the interval has passed, `false` otherwise.

:::note
The timer continues running after the interval has passed. You can call `[reset()](#reset)` to restart it manually. 

:::
